Идея итогового проекта:
Клон Trello (может не полный клон получится, но хотя бы частичный).

По принципу работы:
Приложение для организации своего расписания (что-то на подобии 'to-do-list', только с митинг-бордами). Будет строить борды с тасками на день.
Сами борды можно будет сохранять и модифаить при необходимости (добавлять/удалять, перетаскивать таски).
Попробую также добавить фичу, чтобы можно было сохранять шаблоны и использовать далее при необходимости.

По используемым технологиям / прокачке своих навыков:
- организация маршрутов обработки запросов (Routing);
- технология drag and drop;
- создание новых объектов через классы (борды/доски, списки, карточки);
- обработка и проверка входных данных;
- со стороны клиента: использование локал стореджа (хранение/чтение данных из локального хранилища);
- со стороны сервера: использование БД (хранение/чтение данных из БД).
